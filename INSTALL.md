# TI-Katalog v5 — Інструкція з встановлення

## Що є в цьому архіві

```
ti-v3/
  backend/    ← FastAPI сервер (Python)
  frontend/   ← Next.js 15 сайт
  INSTALL.md  ← ця інструкція
```

---

## Крок 1 — GitHub

1. Створіть новий порожній репозиторій на github.com
2. Завантажте **всі файли** з цього архіву:
   - GitHub → "Add file" → "Upload files"
   - Перетягніть папки `backend/` і `frontend/` разом
   - "Commit changes"

---

## Крок 2 — Railway (Backend + Database)

### 2.1 Створіть проект

1. Відкрийте railway.app → "New Project"
2. "Deploy from GitHub repo" → оберіть ваш репозиторій

### 2.2 Додайте PostgreSQL

1. У проекті: "+ New" → "Database" → "Add PostgreSQL"
2. Дочекайтеся статусу "Running"
3. `DATABASE_URL` додається автоматично — нічого вводити не треба

### 2.3 Налаштуйте Backend сервіс

1. Клікніть на сервіс (не PostgreSQL)
2. **Settings → General → Root Directory:** `backend`
3. **Settings → Variables** → додайте:

| Змінна | Значення |
|--------|---------|
| `PORT` | `8000` |
| `ANTHROPIC_API_KEY` | `sk-ant-...` (з console.anthropic.com) |
| `OPENAI_API_KEY` | `sk-...` (ваш ключ — для семантичного пошуку) |

4. "Deploy" → дочекайтеся "Active"

### 2.4 Скопіюйте URL backend

Виглядає так: `https://ti-katalog-production.up.railway.app`

---

## Крок 3 — Vercel (Frontend)

1. Відкрийте vercel.com → "Add New Project"
2. Оберіть той самий GitHub репозиторій
3. **Root Directory:** `frontend`
4. **Environment Variables** → додайте:

| Змінна | Значення |
|--------|---------|
| `NEXT_PUBLIC_API_URL` | URL з Railway (крок 2.4) |

5. "Deploy" → дочекайтеся "Ready"

---

## Крок 4 — Перший імпорт

1. Відкрийте ваш сайт: `https://ваш-сайт.vercel.app/admin`
2. Введіть Anthropic API Key → "Зберегти"
3. Натисніть **"▶ Імпортувати всі PDF"**
4. Зачекайте ~15-20 хвилин (189 PDF, ~2000 товарів)
5. Статус: 189/189 ✅

## Типові помилки

**Backend не стартує** → перевірте логи Railway. Помилка `DB not ready` нормальна — backend сам повторить 30 разів.

**Vercel build error** → переконайтеся що Root Directory = `frontend` (не root).

**Фото не завантажуються** → нормально до першого імпорту. Після імпорту з'являться автоматично.

**Пошук без 🔮** → OPENAI_API_KEY не додано або порожній. Решта пошуку (SKU + текст) працює без нього.
